## Featured Projects

### Binary Fusion

A low-level Linux binary analysis project focused on ELF64 executable structure, binary validation, compression, and runtime execution.

#### Main features

* Parses and validates Linux ELF64 executables
* Checks architecture, endianness, ABI, file type, and program headers
* Rejects invalid, unsupported, dynamic, or incompatible binaries
* Compresses executable payloads with zlib
* Generates a C launcher stub
* Embeds payloads into a new fused executable
* Works with ELF sections, entry points, and section permissions

**Tech used:**
`Python` `C` `Linux` `ELF64` `LIEF` `GCC` `objcopy` `zlib` `Makefile`

**Repository:** [View project](https://github.com/bruecke0/binary-fusion)

### Encrypt-o-matic

A Windows command-line encryption project built in C# and .NET.
The tool demonstrates secure file encryption, metadata-based recovery, integrity checking, and controlled lab-folder restrictions.

#### Main features

* AES-256-GCM encryption
* ChaCha20-Poly1305 encryption
* Twofish-CBC with HMAC-SHA256
* PBKDF2-SHA256 password verification
* SHA-256 integrity verification
* GZip compression before encryption
* Metadata-based file recovery
* Timer-based unlock using Windows DPAPI
* Recursive directory encryption and decryption

**Tech used:**
`C#` `.NET` `Cryptography` `AES-GCM` `ChaCha20-Poly1305` `Twofish` `PBKDF2` `SHA-256` `Windows`

**Repository:** [View project](https://github.com/bruecke0/encrypt-o-matic)
